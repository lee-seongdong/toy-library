# Member 컨텍스트

## 아키텍처
헥사고날 아키텍처  
의존성 방향: 외부 -> 내부

```
     ┌─────────────────┐
     │   Inbound       │
     │   Adapter       │ ──┐
     └─────────────────┘   │
                           ▼
     ┌─────────────────────────────────┐
     │      Application                │
     │   (Use Cases + Service)         │
     └─────────────────────────────────┘
                           │
                           ▼
     ┌─────────────────────────────────┐
     │        Domain                   │
     │   (Entities + Outbound Ports)   │
     └─────────────────────────────────┘
                           ▲
     ┌─────────────────┐   │
     │   Outbound      │   │
     │   Adapter       │ ──┘
     └─────────────────┘
```
