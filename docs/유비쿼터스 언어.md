# 유비쿼터스 언어
도메인 전문가와 개발자가 공통으로 사용할 용어체계

## 1. 설정 과정
```
용어 식별: 비즈니스 및 개발에서 사용하는 용어들 수집
    ↓
용어 통일: 도메인 전문가와 개발자가 공통으로 사용하는 용어로 통일
    ↓
용어 확정: 유비쿼터스 언어 사전 도출
```


## 2. 산출물
### 2.1. 유비쿼터스 언어 사전
```md
# 도서관 시스템 유비쿼터스 언어 사전

## 핵심 개념
| 용어 | 영문 | 정의 | 예시 |
|------|------|------|------|
| 회원 | Member | 도서관 서비스를 이용할 수 있는 등록된 사용자 | 김철수 회원 |
| 도서 | Book | 대출 가능한 도서관 자료 | 클린 코드 (ISBN: 978-...) |
| 대출 | Loan | 회원이 도서를 일정 기간 빌리는 행위 | 2주간 대출 |
| 반납 | Return | 대출한 도서를 도서관에 돌려주는 행위 | 기한 내 반납 |
| 연체 | Overdue | 반납 기한을 넘긴 상태 | 3일 연체 |
| 예약 | Reservation | 대출중인 도서에 대한 우선권 확보 | 대기 예약 |

## 상태 정의
| 도서 상태 | 설명 |
|-----------|------|
| 대출가능 | Available | 즉시 대출할 수 있는 상태 |
| 대출중 | OnLoan | 다른 회원이 대출한 상태 |
| 예약됨 | Reserved | 예약이 걸린 상태 |
| 정리중 | Processing | 반납 후 정리 중인 상태 |

## 비즈니스 규칙
- 회원은 최대 5권까지 동시 대출 가능
- 대출 기간은 기본 2주, 1회 연장 가능
- 연체 시 하루당 100원의 연체료 부과
- 연체 중인 회원은 추가 대출 불가
```
